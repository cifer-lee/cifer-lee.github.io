PEAR 与 PECL 介绍

PEAR PEAR 全称是 PHP Extension and Application Repository, 和水果 “梨” 的英文发音是相同的. PEAR 存在的目的是: 提供一个有组织结构的开源代码仓库给 PHP 用户们 提供一个代码发布以及包维护的系统 制定一份 PHP 代码风格规范 (在这里: http://pear.php.net/manual/en/standards.php)
